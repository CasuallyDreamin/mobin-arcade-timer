# Mobin's Timer ⏱️

A sleek, production-ready **table timer application** built with **Python** and **PyQt6**.  
Designed for tracking playtime at multiple tables with a professional UI, dark mode styling, and session history logging.  
Originally Developed for my little brother.
---

## ✨ Features

- 🎛️ **Modern Dark UI** with sharp professional design  
- ⏰ **Live Timers** — track multiple tables simultaneously  
- 🗂️ **Session History** — view all past sessions with start/end times  
- 🔄 **Auto-Updating Display** (updates every minute)  
- ⚡ **One-Click Controls** — add new tables and stop timers easily  
- 📜 **Clean Logging** — every session is logged with duration
